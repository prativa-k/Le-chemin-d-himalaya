# Le-chemin-d-himalaya


Le chemin d’Himalaya is my final project for an intensive web development course.
The project was developed from scratch and covers the full development lifecycle,
from design to database and frontend integration.

## Project Overview
The goal of this project was to design and build a complete web application with a
focus on usability, structure, and clear data flow.

## What I Did
- Designed UX/UI wireframes using Figma
- Modeled the database using the Merise methodology
- Designed and implemented a SQL database
- Developed APIs to manage and retrieve data
- Built and integrated the frontend
- Focused on responsive design and user experience

## Technologies Used
- HTML
- CSS
- JavaScript / TypeScript
- SQL
- API development
- Figma (UX/UI design)
- Merise (data modeling)

## What I Learned
- Structuring a project from concept to implementation
- Designing data models and connecting them to a frontend
- Working with APIs and databases
- Building maintainable and user-focused interfaces

## Status
This project is done as part of my course but may be improved further and still have not been finished yet.
